# Production Timeline

This assumes focused 2-hour daily sessions.

## Week 1: Setup and Architecture

Goal: repo setup, docs, naming, stack decisions, GitHub connection.

## Week 2: Scraper Foundation

Goal: browser automation layer, first Forebet fetch, raw snapshot saving.

## Week 3: Scraper UI and Polymarket

Goal: Streamlit scraper UI, source selector, Polymarket first pass.

## Week 4: Parsing and Normalization

Goal: convert raw pages into clean event, market, odds, and prediction records.

## Week 5: PostgreSQL

Goal: schema, inserts, latest records, run history.

## Week 6: Prediction Engine V1

Goal: edge scoring, ranking, Gold/Silver/Bronze signals.

## Week 7: Delivery and Decision Agent

Goal: Telegram publishing, LLM summaries, first decision-agent workflow.

## Week 8: Scraper Agent

Goal: extraction confidence, failure review, selector repair suggestions.

## Week 9: Production Hardening

Goal: tests, CI/CD, deployment notes, monitoring, runbooks.
